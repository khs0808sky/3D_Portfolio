## PlayRule & GameControl
- **게임규칙** <br>
보스몬스터를 공격하여 처치 시 클리어<br><br>
- **조작법** <br>
  
|키|조작|
|:---|:---|
|Mouse|시점 이동|
|W|전방으로 이동|
|A|좌측으로 이동|
|S|후방으로 이동|
|D|우측으로 이동|
|Space|점프|
|LShift|구르기|
|LCtrl|달리기|
|LMouseClick|공격하기|
|RMouseClick|적 쳐다보기(전투 중에 누르는 도중에만)|
|Q|물약 먹기(피가 닳았을 경우)|
|R|재시작하기(게임 클리어 시)|
|X|종료하기(게임 클리어 시)|
